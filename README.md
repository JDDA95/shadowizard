# What is this?

Get perfect shadows every time for the non-designer.

# installation

`npm i shadowwizard --save`

Then...

```
import { shadowizard } from 'shadowizard';

shadowizard({
    shadow_type: 'soft',
    padding: false
})
```

## Options

* *shadow_type* - _hard / soft_ (default to soft)
* *padding* - _boolean (default to false)
